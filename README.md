# Tailwind CSS Playground

An interactive, searchable reference for Tailwind CSS utility classes — built as a single self-contained HTML file with live previews, code tabs, and a dark/light theme.

**Live demo:** [AttilaHuns288452.github.io/tailwind-playground](https://AttilaHuns288452.github.io/tailwind-playground)

## Features

- **35+ interactive sections** — Typography, Colors, Spacing, Flexbox, Grid, Z‑Index, Transforms, Animations, Ring, Backdrop Filters, Shadows, and more
- **Component template library** — 34 pre-built UI components (Navbar, Card, Modal, Pricing Table, Pagination, Toast, Skeleton, Timeline, FAQ, etc.) with clickable style variants
- **🎮 Tailwind Play** — Live REPL: type any Tailwind classes and preview the result in real time. Supports multi-element layouts (one element per line)
- **Dark / Light theme** — One-click toggle, persists via localStorage
- **Search** — Filter sections and components by keyword
- **Code dock** — Inline code viewer with Tailwind + CSS tabs and copy button for every interactive demo
- **Mobile‑friendly** — Collapsible sidebar with smooth navigation

## Tech

- Single `index.html` — no build step, no dependencies
- [Tailwind CSS CDN (Play CDN)](https://tailwindcss.com/docs/installation/play-cdn) — just-in-time CSS generation in the browser
- Vanilla JavaScript — all interactions are plain DOM APIs, no frameworks

## Usage

Open `index.html` in any browser. No server required.

To add new sections or components, edit `index.html` — all data is inline.
