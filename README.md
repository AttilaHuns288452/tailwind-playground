## Why this exists

Most Tailwind references are either a wall of text or a docs site you have to scroll forever. This is the opposite: pick a utility, **see it render instantly**, copy the classes. It's one HTML file you can drop anywhere — bookmark it, fork it, or open it offline.

## What's inside

- **🎮 Tailwind Play REPL** — type any classes and watch them render in real time. One element per line, no setup.
- **36 interactive sections** — Typography, Colors, Spacing, Flexbox, Grid, Border, Shadow, Transform, Animation, Hover, Responsive, Z-index, Ring, Backdrop, Aspect, and more — each with a live preview and copyable code.
- **🧩 Component library** — pre-built UI snippets (Navbar, Card, Modal, Pricing Table, Pagination, Toast, Skeleton, Timeline, FAQ, and more) with clickable style variants.
- **🌓 Dark / Light mode** — one-click toggle, remembered via `localStorage`.
- **🔎 Search** — jump straight to any class, category, or component by keyword.
- **📋 Code dock** — every demo shows its Tailwind + compiled CSS with a one-click copy button.
- **📱 Mobile-friendly** — collapsible sidebar, smooth navigation, works on any screen.

## Tech

- **Single `index.html`** — no framework, no bundler, no dependencies.
- **[Tailwind Play CDN](https://tailwindcss.com/docs/installation/play-cdn)** — just-in-time class generation in the browser.
- **Vanilla JavaScript** — all interactions are plain DOM APIs.

## Usage

1. Open `index.html` in any browser. That's it — no server, no `npm install`.
2. Want it hosted? It's already live on **GitHub Pages**: [attilahuns288452.github.io/tailwind-playground](https://attilahuns288452.github.io/tailwind-playground/)
3. To extend: edit `index.html` — all section and component data is inline.
